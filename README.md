## 汉化介绍

汉化补丁由日不落汉化组基于[umineko-catbox-english](https://andolga.github.io/umineko-catbox-english/)等开源项目制作。
所有翻译文本基于各位译者的翻译底本，并经我们进一步校对、润色而成。感谢几位译者的授权。

### 收录剧本
包含《海猫》《海猫散》《海猫翼·羽·咲》and more...
收录了龙骑士07亲自撰写的，至今最完整《海猫》游戏剧本。

### 汉化特色
- 译文经过进一步的人工校对润色，更加准确与通顺。
- 修正了多处主机版错置的图片·语音·特效，力求呈现更加完美。
- 汉化补丁分为双版本，可以按需选择：
- - “基础版”：保留主机版改动过的文本·语音·BGM
- - “还原版”：多处内容还原为原版文本·语音·BGM

## 本地编译

汉化补丁请前往[官网](https://www.snsteam.club/)下载，此处介绍的是本地编译时的方法。

- 前往[Release页面](https://github.com/Umineko-Project-CN/umineko-catbox-chinese/releases)下载最新资源包，并解压至仓库目录。
- 运行build.bat，将自动生成patch_atoms、patch_yuzu的基础版、还原版汉化补丁，共4版本压缩包。
